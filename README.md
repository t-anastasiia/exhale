<!-- GETTING STARTED -->
## Getting started

### Prerequisites
* Xcode 14+
* Swift 5+
* Firebase
* FirebaseAuth


### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/t-anastasiia/exhale
   ```
2. Find the project folder
   ```sh
   cd exhale/exhale
   ```
3. Launch the app with xcod
   ```sh
   open exhale.xcodeproj
   ```
